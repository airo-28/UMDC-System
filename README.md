# 🏢 UMDC System

A web-based management system for the University of Mindanao Data Center. Designed to organize and track campus data center resources, equipment, and operational workflows.

## Features
- Equipment and resource inventory tracking
- Operational workflow management
- User access control with role-based permissions
- Activity logging and audit trails

## Tech Stack
- **Backend**: PHP
- **Database**: MySQL
- **Frontend**: HTML, CSS, JavaScript

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/airo-coder/UMDC-System.git
   cd UMDC-System
   ```

2. Import the database:
   - Create a MySQL database
   - Import the included SQL file

3. Configure the database connection:
   - Update database credentials in the config file

4. Start a local server:
   ```bash
   php -S localhost:8000
   ```

5. Open [http://localhost:8000](http://localhost:8000) in your browser.
